# Portfolio Flutter

Application Flutter simple pour un portfolio personnel — responsive et facile à personnaliser.

Remplacement rapide:
1. Créez un nouveau projet Flutter (ou utilisez votre projet existant).
2. Remplacez le contenu de `lib/main.dart` par le fichier fourni.
3. Copiez `pubspec.yaml` ou ajoutez la dépendance `url_launcher` puis lancez `flutter pub get`.

Commandes pour exécuter (PowerShell):

```powershell
flutter pub get
flutter run -d chrome    # pour web
flutter run               # pour mobile connecté
```

Personnalisation:
- Modifiez votre nom, descriptions et les projets dans `lib/main.dart`.
- Remplacez l'image de profil par votre propre URL ou ajoutez `assets` et référencez-les dans `pubspec.yaml`.

Besoin d'aide pour personnaliser le design ou ajouter des pages ? Dites-moi ce que vous voulez changer.
